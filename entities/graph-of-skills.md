# Graph of Skills

## Overview
Graph of Skills retrieves bounded, dependency-aware bundles from large skill libraries instead of treating skill recall as flat semantic search. It assumes success creates a library problem as surely as failure creates a learning problem.

## Why it matters
It matters because once a system has many learned skills, retrieval becomes structural. A giant undifferentiated skill heap is only a slower form of forgetting.

## Distinctive trait
Its distinctive trait is dependency-aware structural retrieval: the retrieved bundle reflects how skills compose, not just which title looks semantically nearby.
