# WebShop

## Overview
WebShop is a scalable e-commerce interaction environment with real products, crowd-sourced instructions, and both RL and imitation-learning baselines. It predates the current browser-gym layer but remains a strong early example of grounded web interaction for agents.

## Why it matters
It matters because it showed that one could build a task environment rich enough for strategy, re-formulation, and exploration before the field had fully realistic browser worlds.

## Distinctive trait
Its distinctive trait is being an early web interaction world with enough realism to support learning, yet enough structure to remain usable.
