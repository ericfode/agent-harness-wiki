# SkillFoundry

## Overview
SkillFoundry builds validated agent skill packages from heterogeneous resources such as repositories, APIs, scripts, notebooks, documentation, databases, and papers. It treats external resources as ore from which executable skill artifacts can be smelted, verified, and maintained.

## Why it matters
It matters because it makes the skill library an active knowledge-compilation pipeline rather than a static folder of hand-written tricks. That is much closer to a real long-lived learning substrate.

## Distinctive trait
Its distinctive trait is validated skill packaging with provenance and tests across many resource types, followed by iterative expansion, repair, and pruning.

## Relationships
Read SkillFoundry with [[memento-skills]], [[evoskills|EvoSkills]], and [[skillx|SkillX]]. It is also a concrete embodiment of the writable-artifact thesis behind [[self-evolving-workflows]].
