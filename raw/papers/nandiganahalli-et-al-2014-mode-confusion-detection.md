# User Interface Validation using Mode Confusion Detection

**Source:** [DOI](https://doi.org/10.2514/6.2014-2349)
**Authors:** Jayaprakash Suraj Nandiganahalli, Sangjin Lee, Inseok Hwang, Bong-Jun Yang
**Date:** 2014

## Core idea
The paper models interface state, operator expectation, and automation mode to detect mismatches that lead to dangerous mode confusion.

## Harness takeaway
Live, replay, simulation, draft-approval, and branch-inspection modes should be modeled explicitly and validated, not merely color-coded after the fact.
