# Analyst's Workspace

**Source:** [DOI](https://doi.org/10.1109/VAST.2012.6400559)
**Authors:** Christopher Andrews, Chris North
**Date:** 2012

## Core idea
Analyst's Workspace uses a large high-resolution display to let analysts externalize sensemaking through spatial arrangement of documents, clues, and hypotheses.

## Harness takeaway
Spatial arrangement of evidence, branches, and hypotheses should be persisted as real analysis state in the studio, not treated as disposable window choreography.
