# Rethinking Agent Design: From Top-Down Workflows to Bottom-Up Skill Evolution

**Source:** [arXiv](https://arxiv.org/abs/2505.17673)
**Authors:** Jiawei Du, Jinlong Wu, Yuzheng Chen, Yucheng Hu, Bing Li, Joey Tianyi Zhou
**Date:** 2025-05-23
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
This paper argues against a purely top-down workflow-authoring view of agents and instead emphasizes skill acquisition through experience, reflection, and sharing. It is a useful conceptual foil for workflow-generation papers because it asks whether reusable skills, not only reusable graphs, are the true long-term substrate.
