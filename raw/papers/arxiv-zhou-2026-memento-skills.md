# Memento-Skills: Let Agents Design Agents

**Source:** [arXiv](https://arxiv.org/abs/2603.18743)

## Abstract
Memento-Skills presents a continual-learning agent system that keeps model weights fixed and shifts adaptation into an external skill memory. The paper's central objects are stateful prompts and reusable skills stored as structured markdown artifacts, with a read-execute-reflect-write loop that retrieves skills, runs them, and then revises or extends the library from experience. The authors position this as an agent that can design task-specific agents rather than only execute a static, human-authored toolkit. Reported benchmark results on GAIA and Humanity's Last Exam improve across reflective learning rounds, with relative gains of 26.2% and 116.2% respectively. The paper links the full implementation at the companion GitHub repository.
