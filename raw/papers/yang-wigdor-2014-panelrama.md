# Panelrama

**Source:** [DOI](https://doi.org/10.1145/2556288.2557199)
**Authors:** Jishuo Yang, Daniel Wigdor
**Date:** 2014

## Core idea
Panelrama is a web framework for distributed user interfaces with built-in UI-state synchronization and dynamic allocation across devices based on device characteristics.

## Harness takeaway
A moldable operations studio should treat surfaces as reassignable projections over one shared operational state, not as separate clients with private truth.
