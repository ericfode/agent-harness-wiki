# Terrarium: Revisiting the Blackboard for Multi-Agent Safety, Privacy, and Security Studies

**Primary category:** cs.AI
**All categories:** cs.AI, cs.CL, cs.CR

## Abstract
Terrarium repurposes the blackboard pattern as a configurable testbed for safety, privacy, and security studies in LLM-based multi-agent systems. The important move is architectural: by routing collaboration through an explicit shared substrate, the framework makes misalignment, malicious agents, compromised communication, and data-poisoning attacks easier to model and isolate. For this wiki, it strengthens the case that shared-workspace coordination is useful not only for productivity but also for adversarial analysis.
