# MemSkill: Learning and Evolving Memory Skills for Self-Evolving Agents

**Source:** [arXiv](https://arxiv.org/abs/2602.02474)
**Authors:** Haozhen Zhang, Quanyu Long, Jianzhu Bao, Tao Feng, Weizhi Zhang, Haodong Yue, Wenya Wang
**Date:** 2026-02-02
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
MemSkill turns memory operations themselves into evolvable skills chosen by a controller and periodically revised by a designer. It matters because it makes even the memory-maintenance layer part of the self-evolving workflow story rather than leaving it as fixed plumbing.
