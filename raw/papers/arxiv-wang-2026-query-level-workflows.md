# Do We Always Need Query-Level Workflows? Rethinking Agentic Workflow Generation for Multi-Agent Systems

**Source:** [arXiv](https://arxiv.org/abs/2601.11147)
**Authors:** Zixu Wang, Bingbing Xu, Yige Yuan, Huawei Shen, Xueqi Cheng
**Date:** 2026-01-16
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
This paper is a useful counterpoint to the assumption that every query deserves a freshly generated workflow. It argues that a small set of strong task-level workflows can often cover many queries at much lower token cost, which is highly relevant when deciding how often a control plane should evolve workflows online rather than promote a stable repertoire.
