# LLM Agent Communication Protocol (LACP) Requires Urgent Standardization: A Telecom-Inspired Protocol is Necessary

**Source:** [arXiv](https://arxiv.org/abs/2510.13821)
**Authors:** Xin Li, Mengbing Liu, Chau Yuen
**Date:** 2025-09-26
**Primary category:** cs.NI
**All categories:** cs.NI

## Abstract
This position paper argues that agent coordination is already hitting the same fragmentation problem early telecom systems faced before protocol standardization. It proposes a layered communication model meant to provide semantic clarity, transactional integrity, and built-in security for LLM agents. Even if the specific stack changes, the paper is a useful marker that typed protocol design remains early and underconsolidated compared with the volume of ad hoc message-passing systems.
