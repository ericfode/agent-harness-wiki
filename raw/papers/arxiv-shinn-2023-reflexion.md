# Reflexion: Language Agents with Verbal Reinforcement Learning

**Source:** [arXiv](https://arxiv.org/abs/2303.11366)
**Authors:** Noah Shinn, Federico Cassano, Edward Berman, Ashwin Gopinath, Karthik Narasimhan, Shunyu Yao
**Date:** 2023-03-20
**Primary category:** cs.AI
**All categories:** cs.AI, cs.CL, cs.LG

## Abstract
Reflexion lets language agents learn from trial-and-error via linguistic feedback rather than weight updates. It stores reflective text in episodic memory and uses that memory to improve subsequent decisions across coding and reasoning tasks. The paper is not yet about workflow artifacts proper, but it is a foundational precursor for systems that write durable lessons back into external state.
