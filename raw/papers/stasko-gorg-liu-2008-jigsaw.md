# Jigsaw: Supporting Investigative Analysis through Interactive Visualization

**Source:** [DOI](https://doi.org/10.1057/palgrave.ivs.9500180)
**Authors:** John Stasko, Carsten Görg, Zhicheng Liu
**Date:** 2008

## Core idea
Jigsaw coordinates multiple linked views over documents and entities so analysts can form and test hypotheses by moving between evidence representations.

## Harness takeaway
A studio evidence workspace should coordinate entity, document, timeline, branch, and claim views rather than trapping each artifact in its own pane.
