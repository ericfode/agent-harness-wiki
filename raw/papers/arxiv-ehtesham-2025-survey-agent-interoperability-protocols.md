# A survey of agent interoperability protocols: Model Context Protocol (MCP), Agent Communication Protocol (ACP), Agent-to-Agent Protocol (A2A), and Agent Network Protocol (ANP)

**Source:** [arXiv](https://arxiv.org/abs/2505.02279)
**Authors:** Abul Ehtesham, Aditi Singh, Gaurav Kumar Gupta, Saket Kumar
**Date:** 2025-05-04
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
This survey maps four protocol families that are beginning to structure agent interoperability: MCP for typed tool access, ACP for general structured messaging, A2A for peer delegation, and ANP for decentralized discovery and collaboration. Its main value is comparative rather than prescriptive. It helps separate several problems that are often blurred together in agent papers: tool invocation, session-aware messaging, peer task delegation, and open-network agent discovery.
