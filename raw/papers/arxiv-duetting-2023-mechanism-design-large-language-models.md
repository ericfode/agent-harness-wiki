# Mechanism Design for Large Language Models

**Source:** [arXiv](https://arxiv.org/abs/2310.10826)
**Authors:** Paul Duetting, Vahab Mirrokni, Renato Paes Leme, Haifeng Xu, Song Zuo
**Date:** 2023-10-16
**Primary category:** cs.GT
**All categories:** cs.GT, econ.TH

## Abstract
This is not a harness paper, but it matters for coordination because it treats LLM-mediated generation as an allocation problem with incentive constraints. The paper defines a token-level auction where LLM agents express preferences over stochastic outputs and shows when monotonicity is sufficient for second-price style mechanisms. For agent runtimes, it is evidence that work routing and output shaping can be framed as priced coordination rather than left as informal planner heuristics.
