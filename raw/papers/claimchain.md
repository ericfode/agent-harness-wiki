# ClaimChain: Improving the Security and Privacy of In-band Key Distribution for Messaging

**Source:** [arXiv](https://arxiv.org/abs/1707.06279)
**Authors:** Wouter Lueks, Marios Isaakidis, George Danezis, Carmela Troncoso
**Date:** 2017

## Abstract / key passage
ClaimChain uses authenticated append-only chains of claims and beliefs so users can publish verifiable information about themselves and selectively share claims about others without exposing the entire trust graph.

## Harness takeaway
This is very close to a sovereignty-preserving harness identity layer: each node can keep a signed history of claims and cross-vouching without collapsing into one global directory or one public reputation scoreboard.
