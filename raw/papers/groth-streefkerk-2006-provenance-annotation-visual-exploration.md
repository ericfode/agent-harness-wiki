# Provenance and Annotation for Visual Exploration Systems

**Source:** [DOI](https://doi.org/10.1109/TVCG.2006.101)
**Authors:** D.P. Groth, K. Streefkerk
**Date:** 2006

## Core idea
Exploration history and annotations can be captured together so analysts can revisit, explain, and share how a result was obtained rather than only the final result.

## Harness takeaway
The studio should preserve replayable evidence histories with annotations, bookmarks, and named analytic steps rather than static screenshots or transcript snippets.
