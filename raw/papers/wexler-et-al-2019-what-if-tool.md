# The What-If Tool

**Source:** [DOI](https://doi.org/10.1109/TVCG.2019.2934619)
**Authors:** James Wexler, Mahima Pushkarna, Tolga Bolukbasi, Martin Wattenberg, Fernanda Viegas, Jimbo Wilson
**Date:** 2019

## Core idea
The What-If Tool supports interactive counterfactual editing, subset comparison, and probing of model behavior without requiring bespoke coding.

## Harness takeaway
A moldable operations studio should support direct counterfactual editing over inputs, assumptions, or branch conditions with immediate visible consequences.
