# G2CP: A Graph-Grounded Communication Protocol for Verifiable and Efficient Multi-Agent Reasoning

**Source:** [arXiv](https://arxiv.org/abs/2602.13370)
**Authors:** Karim Ben Khaled, Davy Monticolo
**Date:** 2026-02-13
**Primary category:** cs.MA
**All categories:** cs.MA, cs.AI, cs.CL

## Abstract
G2CP argues that many multi-agent failures come from letting agents communicate in unrestricted natural language. It replaces free-text exchanges with graph operations over a shared knowledge graph, which makes messages auditable, reduces semantic drift, and gives verifiable reasoning traces. The reported gains in token efficiency and task accuracy make this one of the clearest recent examples of structural, typed communication as a coordination primitive rather than an implementation detail.
