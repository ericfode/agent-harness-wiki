# Trace2Skill: Distill Trajectory-Local Lessons into Transferable Agent Skills

**Source:** [arXiv](https://arxiv.org/abs/2603.25158)
**Authors:** Jingwei Ni, Yihao Liu, Xinpeng Liu, Yutao Sun, Mengyu Zhou, Pengyu Cheng, Dexin Wang, Erchao Zhao, Xiaoxi Jiang, Guanjun Jiang
**Date:** 2026-03-26
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
Trace2Skill consolidates trajectory-local lessons into unified transferable skills by using multiple sub-agents to analyze broad execution experience before distillation. It is especially relevant to the problem of turning many local workflow failures into one durable procedural improvement rather than a pile of brittle patches.
