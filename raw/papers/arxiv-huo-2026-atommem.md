# AtomMem : Learnable Dynamic Agentic Memory with Atomic Memory Operation

**Source:** [arXiv](https://arxiv.org/abs/2601.08323)
**Authors:** Yupeng Huo, Yaxi Lu, Zhong Zhang, Haotian Chen, Yankai Lin
**Date:** 2026-01-13
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
AtomMem decomposes memory management into atomic CRUD operations and learns a policy over that memory workflow. It is adjacent rather than identical to workflow evolution, but it is a strong example of a previously static control routine being turned into a learnable policy.
