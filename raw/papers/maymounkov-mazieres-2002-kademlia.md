# Kademlia: A Peer-to-Peer Information System Based on the XOR Metric

**Source:** [Springer](https://link.springer.com/chapter/10.1007/3-540-45748-8_5)
**Authors:** Petar Maymounkov, David Mazières
**Date:** 2002-10-10

## Abstract / key passage
Kademlia describes a peer-to-peer distributed hash table with fault-tolerant lookup and routing based on an XOR metric. Its main value is giving a decentralized network an efficient way to discover nodes and locate information without requiring one central broker.

## Harness takeaway
If a giant harness network becomes genuinely peer-to-peer, it needs a discovery and rendezvous substrate. Kademlia is one of the clean canonical answers for locating peers, spaces, or replicas without routing everything through a permanent central coordinator.
