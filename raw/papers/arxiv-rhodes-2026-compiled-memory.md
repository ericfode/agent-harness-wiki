# Compiled Memory: Not More Information, but More Precise Instructions for Language Agents

**Source:** [arXiv](https://arxiv.org/abs/2603.15666)
**Authors:** James Rhodes, George Kang
**Date:** 2026-03-12
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
Compiled Memory studies a different memory question: not how to retrieve more prior information, but how to convert experience into more precise instructions. Its Atlas kernel verifies facts from successes and failures through a promotion gate and rewrites the agent instruction structure rather than only injecting more context. The paper is a strong precedent for turning experience into edited operating procedure.
