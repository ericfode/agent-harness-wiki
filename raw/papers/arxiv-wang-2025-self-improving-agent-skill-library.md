# Reinforcement Learning for Self-Improving Agent with Skill Library

**Source:** [arXiv](https://arxiv.org/abs/2512.17102)
**Authors:** Jiongxiao Wang, Qiaojing Yan, Yawei Wang, Yijun Tian, Soumya Smruti Mishra, Zhichao Xu, Megha Gandhi, Panpan Xu, Lin Lee Cheong
**Date:** 2025-12-18
**Primary category:** cs.AI
**All categories:** cs.AI

## Abstract
This paper introduces SAGE, an RL framework where agents accumulate and use skills across sequential rollouts. It is valuable because it pushes skill-library self-improvement beyond prompt-only methods and makes the promotion of new skills part of the training loop rather than an occasional manual ceremony.
