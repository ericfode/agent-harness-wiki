# From common operating picture to situational awareness

**Source:** [DOI](https://doi.org/10.1504/IJEM.2014.061659)
**Authors:** Erna Danielsson, Aida Alvinius, Gerry Larsson
**Date:** 2014

## Core idea
A common operating picture is a continuously updated shared snapshot, while role-specific situational awareness depends on local goals, role knowledge, and interpretation.

## Harness takeaway
The studio should maintain one canonical operational picture but derive role-filtered situational panes for oncall, reviewer, planner, and investigator.
