# Code space

**Source:** [DOI](https://doi.org/10.1145/2076354.2076393)
**Authors:** Andrew Bragdon, Rob DeLine, Ken Hinckley, Meredith Ringel Morris
**Date:** 2011

## Core idea
Code Space studies co-located programming around a shared display plus personal devices, emphasizing public/private territories and promotion of local work into the shared room context.

## Harness takeaway
A wallboard should be the public operational picture, while personal devices stage filters, notes, and candidate actions that are explicitly promoted into shared view.
