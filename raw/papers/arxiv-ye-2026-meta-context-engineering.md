# Meta Context Engineering via Agentic Skill Evolution

**Source:** [arXiv](https://arxiv.org/abs/2601.21557)
**Authors:** Haoran Ye, Xuning He, Vincent Arak, Haonan Dong, Guojie Song
**Date:** 2026-01-29
**Primary category:** cs.AI
**All categories:** cs.AI, cs.NE

## Abstract
Meta Context Engineering co-evolves context-engineering skills and context artifacts instead of relying on fixed reflection loops or hand-designed context schemas. It is especially relevant where self-evolving workflows start to blur into self-evolving instruction surfaces.
