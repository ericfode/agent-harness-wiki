# Gas Town: From Clown Show to v1.0

## Beads v1.0.0
- Drop-in, unopinionated memory system and knowledge graph
- Tracks the "Why" — design intent behind changes
- Work items stored as JSONL in Git, linked as multi-graph, queryable via SQL
- Moved from SQLite+JSONL to Dolt (Git for data)
- 20k+ GitHub stars

## Gas Town v1.0.0
- 13k stars
- The Mayor abstraction: concierge interface, Chief of Staff for your agent swarm
- "If Claude Code is an Executive Assistant, Gas Town Mayor is your Chief of Staff"

## Gas City (alpha)
- Modular platform: decomposes the stack into primitives
- Primitives: Identity, roles, messaging, cost tracking, multi-model dispatch, molecules, seances
- Wasteland: federation protocol using Dolt and Git
- End state: talking to a visual interface (a "face") rather than typing
