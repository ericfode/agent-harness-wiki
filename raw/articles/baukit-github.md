# baukit

**Source:** [David Bau](https://github.com/davidbau/baukit)
**Date:** unknown

## Summary
GitHub repository for David Bau’s lightweight toolkit for tracing and intervening in PyTorch models. It has been a common practical substrate for causal tracing and model editing work.

## Why it matters here
This is useful as a representative of the lower-level intervention tooling ecosystem beneath more polished libraries. In practice, NNPL experiments will rely on exactly this sort of plumbing.
