# doug

**Source:** [Eilene Tomkins-Flanagan](https://github.com/eilene-ftf/doug)
**Date:** unknown

## Summary
Repository for the Doug polynomial-time programming language defined over a vector-symbolic architecture. It is a concrete implementation-level reference point for the typed-VSA direction discussed in the draft.

## Why it matters here
The research value here is not merely that Doug exists, but that it treats a vector-symbolic substrate as a language design surface rather than as a post hoc embedding trick. That makes it unusually close in spirit to NNPL.
