# Gas Town Emergency User Manual

**Source:** [Steve Yegge on Medium](https://steve-yegge.medium.com/gas-town-emergency-user-manual-cf0e4556d74b)
**Date:** January 13, 2026
**Topic:** Operational guidance for actually running Gas Town crews.

---

## 1. Convoys as tracked swarm work
Yegge describes convoy-based work as the practical way to sling well-specified beads and epics through the dashboard while tracking feature progress.

## 2. PR Sheriff as review governance
The manual introduces a PR Sheriff role with standing orders to triage open pull requests, separate easy wins from human-review cases, and route merge/cleanup work to the crew.

## 3. Swarm operations as explicit procedure
The document reads less like a philosophy essay and more like an operating manual, which strengthens the case that Gas Town is a real work-definition and routing system rather than merely a colorful metaphor.
