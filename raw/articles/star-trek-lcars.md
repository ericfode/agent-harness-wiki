# LCARS

**Source:** [Wikipedia](https://en.wikipedia.org/wiki/LCARS)
**Topic:** A shared visual grammar spanning ship consoles, wall displays, and handheld interfaces.

## Core idea
LCARS is interesting not as colorful touchscreen ornament but as one consistent interaction grammar rendered across many surfaces and roles.

## Harness takeaway
A moldable operations studio should let one shared state model appear as a wallboard, queue item, inspector, or CLI reference without changing the underlying truth.
