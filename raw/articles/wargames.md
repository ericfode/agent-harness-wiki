# WarGames

**Source:** [Wikipedia](https://en.wikipedia.org/wiki/WarGames)
**Topic:** Room-scale shared situation display and dangerous ambiguity between simulation and live operations.

## Core idea
WarGames is a warning that a command center needs explicit mode clarity: simulation, replay, and live operation must never blur.

## Harness takeaway
A harness should make live, replayed, simulated, and inferred state visibly distinct with hard mode markers and durable acknowledgment trails.
