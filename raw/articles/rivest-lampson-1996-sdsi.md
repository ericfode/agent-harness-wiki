# SDSI - A Simple Distributed Security Infrastructure

**Source:** [SDSI](https://people.csail.mit.edu/rivest/pubs/RL96.ver-1.1.html)
**Authors:** Ronald L. Rivest, Butler Lampson
**Date:** 1996-10-02

## Core idea
SDSI emphasizes linked local namespaces and group-membership certificates rather than assuming one universal global naming hierarchy.

## Harness takeaway
Composite identity in the studio should be expressed as linked local names and memberships rather than one flattening global actor label.
