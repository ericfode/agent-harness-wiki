# Prime Radiant

**Source:** [Foundation Wiki](https://foundation.fandom.com/wiki/Prime_Radiant)
**Topic:** A manipulable artifact containing forecasts, branches, and model state.

## Core idea
The Prime Radiant works as an inspectable model object rather than as hidden backend machinery. Forecasts and branching futures are accessed through a shared artifact.

## Harness takeaway
A harness should treat model and counterfactual inspection as first-class artifacts with provenance and authority boundaries, not merely as backend implementation details.
