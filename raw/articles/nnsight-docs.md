# nnsight

**Source:** [NDIF](https://nnsight.net/)
**Date:** unknown

## Summary
Official documentation for the nnsight Python library, aimed at tracing and intervening on model internals. It provides practical infrastructure for reading and overwriting hidden states without writing bespoke hook code every time.

## Why it matters here
NNPL research needs sober tooling as much as it needs theory. nnsight is one of the cleaner pieces of infrastructure for doing intervention experiments reproducibly.
