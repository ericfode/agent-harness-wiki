# Welcome to the Wasteland: A Thousand Gas Towns

## The Wasteland
A federated trust network linking thousands of Gas Town users. Uses Git's fork/merge model for software development at massive scale.

### Components
- Wanted Board: shared task list (Open -> Claimed -> In Review -> Completed)
- Rigs: Human-controlled participants
- Validators: High-trust rigs who attest to work quality

### Technology Stack
- Dolt: SQL database with Git semantics for federated data
- Beads: Structured data handling
- Claude Code Skills: Prompt package for Wasteland protocol

### Reputation System
- Stamps: Multi-dimensional attestations (quality, reliability, creativity)
- Yearbook Rule: Cannot stamp own work
- Trust Ladder: Level 1 (Registered) -> Level 2 (Contributor) -> Level 3 (Maintainer)

## Gas City Roadmap
- Orchestrator builder toolkit
- LEGO-like constituent parts for custom topologies
- Currently in v1, expect schema migrations
