# Ancillary Justice

**Source:** [Wikipedia](https://en.wikipedia.org/wiki/Ancillary_Justice)
**Topic:** Composite identity distributed across multiple embodiments.

## Core idea
Ancillary Justice explores a mind spanning many bodies, where partitions and embodiment changes affect what the same actor can know and do.

## Harness takeaway
A harness should model composite actor identity with embodiment-specific sessions and explicit causal divergence rather than assuming one seamless agent everywhere.
